このファイルには、以下の書籍のコードスニペットのリストが含まれます。
	
	Bayes’ Rule With Python by JV Stone.
	（日本語タイトル：［速習］ベイズの定理 ——「推論」に効く数学の力）

本書の第１章は以下のURLからダウンロード可能です。
	
	http://jim-stone.staff.shef.ac.uk/BookBayes2012/BayesRuleBookMain.html

また、以下のコードは下記URLからダウンロード可能です。

	PYTHON: 	http://jim-stone.staff.shef.ac.uk/BookInfoTheory/InfoTheoryPython.html
	MATLAB: 	http://jim-stone.staff.shef.ac.uk/BookInfoTheory/InfoTheoryMatlab.html
	R: 			http://jim-stone.staff.shef.ac.uk/BookInfoTheory/InfoTheoryR.html

インポートが必要なライブラリ: Numpy, matplotlib（訳注：左記以外にも、statsmodelsのインストールが必要です）
また、コードはPython 3.5を用いてテストを行いました。

各ファイルの名前は、本書の中で関連している図、式、もしくはテーブルの番号を指しています。
番号　ファイル名		

Chapter 1

1.2			Ch1Eq09.py		水疱瘡の事後確率を推定します。
1.3			Ch1Eq26.py		ベイズ因子と事前オッズを掛け合わせることで、事後オッズを推定します。
1.4			Ch1Eq34.py		フォー・キャンドルとフォークハンドルの問題。事後確率を推定します。

	なし

3.1			Ch3Fig2a.py		同時確率を描画。図3.2aの離散確率分布の散布図を描画します。
3.2			Ch3Q1.py		問い１に答えるために同時確率の値を求めます。
3.3			Ch3Q2.py		問い２に答えるために周辺尤度を求めます。
3.4			Ch3Q3.py		問い３に答えるために事前確率分布を求めます。
3.5			Ch3Q4.py		問い４に答えるために尤度関数を求めます。
